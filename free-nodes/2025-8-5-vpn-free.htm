很多用户初次听到这个名字，可能会将“Kali”误认为是Kali Linux的衍生品，事实上这里的“Kali Shadowrocket”是 Shadowrocket 的定制版本，在应用分发平台中存在多个类似命名的变体，但其核心功能仍以多协议支持、稳定连接、隐私保护为主。

你可能会关心以下几个问题：

  • 它支持哪些协议？

  • 配置是否复杂？

  • 付费值得吗？

答案是：它非常值得拥有，尤其适合注重体验的iOS用户。

二、Kali Shadowrocket核心功能解析

1. 多协议支持

支持包括 Shadowsocks、Vmess、Trojan、Socks5、HTTP、WireGuard 在内的多种主流代理协议，几乎覆盖市面上所有主流的科学上网节点类型。

2. 灵活的规则配置

你可以根据自己的需求，自定义分流规则、应用过滤、海外IP策略等。比如，设置只有国外网站通过代理，而国内网站保持直连，有效减少不必要的资源占用。

3. 数据加密传输

全程使用安全的加密方式传输数据，即使你在咖啡馆、机场、酒店等开放 Wi-Fi 环境中上网，也能保障隐私不被窥探。

4. 实时日志与连接统计

应用内提供详细的连接日志、节点测速、实时带宽监控等功能，帮助用户随时掌控网络状态，定位问题更加轻松。

三、安装 Kali Shadowrocket 的正确姿势

注意：Kali Shadowrocket 是一款付费应用，需使用非中国大陆 Apple ID 登录 App Store 进行购买与下载。

  1. 打开 App Store，点击头像退出当前账号；

  2. 使用 美区、港区或其他支持该应用的 Apple ID 登录；

  3. 可通过朋友提供，或在网络平台寻找共享账号（注意账号安全）；

  1. 搜索 “Shadowrocket” 或 “Kali Shadowrocket”（不同分发版本命名可能略有不同）；

  2. 应用图标一般是一个火箭形状，价格在 $2.99 - $4.99 美元之间；

  3. 完成支付后点击“获取”开始下载安装；

四、Kali Shadowrocket 的配置详解

安装完毕后，首次打开 Kali Shadowrocket，会出现权限申请与系统 VPN 配置提示，按需确认。

1. 添加节点配置

方法一：手动输入

  1. 打开 App 主界面；

  2. 点击右上角 “+” 号，选择协议类型（Shadowsocks / Vmess / Trojan 等）；

  3. 输入服务器地址、端口、密码/UUID、加密方式等；

  4. 命名并保存配置；

方法二：扫码添加

  1. 获取二维码（可通过订阅服务商、TG 群组等获取）；

  2. 点击 “扫码” 图标；

  3. 扫描后自动导入节点信息；

方法三：订阅链接导入

  1. 获取订阅链接（一般格式为 https://.../sub）；

  2. 在设置页面点击 “订阅”；

  3. 添加链接并命名；

  4. 可设为自动更新，便于节点维护；

2. 设置代理规则

Kali Shadowrocket 提供强大的规则管理能力，默认规则如下几种：

  • 全局模式：所有流量走代理；

  • 直连模式：所有流量不通过代理；

  • 规则模式：由预设规则决定哪些走代理，哪些直连；

  • 绕过大陆与局域网：常见分流方案，智能识别 IP；

你还可以从 GitHub 等平台下载专业的 规则文件，如 ACL4SSR、ConnersHua 的配置，导入到 Kali Shadowrocket 实现更精细化控制。

3. 启动代理服务

  • 在主界面切换你需要使用的节点；

  • 点击右上角的“开关”按钮；

  • 状态栏出现 VPN 标志即表示连接成功；

五、实用技巧与注意事项

技巧一：测速选择最优节点

在订阅节点列表页面，长按节点名称，选择“测速”，可以快速识别当前网络条件下最快的服务器。

技巧二：区分应用代理

你可以指定某些 App 走代理，某些 App 直连（如淘宝、微信），避免不必要的干扰。

路径为：配置 → 策略 → 分应用代理

技巧三：开启 IPv6 支持

若运营商支持 IPv6，Shadowrocket 也可以针对 IPv6 流量进行控制，提升访问效率。
